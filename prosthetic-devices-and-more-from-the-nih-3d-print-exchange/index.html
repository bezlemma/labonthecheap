Prosthetic Devices and More from the NIH 3D Print Exchange

June 28, 2018

We'd like to use the Prosthetic Devices curated by e-NABLE to plug the NIH 3D Print Exchange for a second time. These 3D-printable prosthetics allow for DIY assistive devices that can be easily customized and printed.

The NIH 3D Print Exchange also has a wide variety of 3D printable labware, some of which we've covered in previous posts, like this nifty magnetic tube rack for DNA extraction.
